## Notes

- **Code source:** The `.tsx` files are React + TypeScript + Tailwind CSS + daisyUI reference implementations aligned with the Figma daisyUI library. Adapt class names and structure to match your project (e.g. shadcn/ui or your design tokens).
- **Figma → code:** For pixel-perfect code from Figma, use **Figma Dev Mode** (Inspect panel) or **Figma’s “Copy as CSS/SVG”** on the specific frames. The Figma MCP was used to map nodes; full generated code may need to be pulled per screen from Figma.
- **Tailwind + daisyUI:** Use [daisyUI](https://daisyui.com/) with Tailwind. Example: `btn btn-primary`, `card`, `modal`, `alert`, etc.
