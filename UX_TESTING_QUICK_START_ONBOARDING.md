## 🎯 Common Issues to Watch

### Issue 1: Widget Not Noticed
**Symptom**: User doesn't see the profile completion widget  
**Check**: Is it prominent enough? Right placement?  
**Fix**: Increase visibility, add animation, move to top

### Issue 2: Unclear Gating
**Symptom**: User doesn't understand why services are locked  
**Check**: Is message clear? Does it explain requirements?  
**Fix**: Improve messaging, add examples

### Issue 3: Fields Hard to Find
**Symptom**: User can't find profile fields  
**Check**: Is navigation clear? Are tabs obvious?  
**Fix**: Improve navigation, add breadcrumbs

### Issue 4: Mobile Issues
**Symptom**: Onboarding/widget broken on mobile  
**Check**: Test on actual device  
**Fix**: Responsive design fixes
